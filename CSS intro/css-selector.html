What is CSS selector?

CSS selector are the element that is used to select the parts of HTML to design or decorate it.

Types of CSS Selector

  1. Element Selector
  2. Class Selector
  3. ID Selector
  4. Child Selector
  5. Descendant Selector
  6. Pesudo Selector
